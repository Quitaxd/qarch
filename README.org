* qarch
An Arch Linux repo for everyone!

** Installation
Add these lines into your pacman.conf

#+BEGIN_SRC
[qarch]
SigLevel = Optional TrustAll
Server = https://github.com/Quitaxd/qarch/raw/main/$arch
#+END_SRC
